# Bank System Console Application (C++)

A fully functional C++ console-based Bank System that simulates real-world banking operations.  
This system allows admins and users to manage clients, perform transactions, control permissions, and handle user accounts efficiently through an intuitive console interface.

---

## ✨ Features

- **User Authentication:** Secure login system for admins and users.  
- **Client Management:** Add, update, delete, and find clients.  
- **User Management:** Add, update, delete, and find system users.  
- **Transactions:** Deposit and withdraw money for clients.  
- **Permissions Control:** Users have specific permissions to perform operations.  
- **Data Storage:** Client and user data stored in text files (`Clients.txt`, `Users.txt`).  
- **Console UI:** Interactive menus for easy navigation.  
- **Input Validation:** Prevents invalid data entry for safer operations.
